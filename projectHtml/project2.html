Grocery Store Android Application

Used Languages: Java & SQLite

Introducing a user-friendly Android software that enables users to purchase goods from nearby stores online. The main goal of this project is to give small businesses a platform to communicate with clients through the application, broaden their customer base, and boost sales. Users get immediate access to the groceries they require from the comfort of their homes with a seamless, intuitive interface, ensuring a convenient and pleasurable shopping experience.

In the login page, the User has to enter his registered username and password. If the username or password is incorrect, the user will not be able to log in. If the username and password are correct then it is directed to the next page. According to what the user has registered further screens will be opened. On the login, screen user will also have the option “Sign Up” to register a new account.

A new user will have to register in the application by delivering the necessary details to get the vision of the items in the system. The user has to fill in his details like name, phone number, email address, address, etc. The user has to set a username and a password which will be used for login. If the user already has an account then in the register screen user will also have an option “Sign In” to log in to the old account.

In the Grocery Home page, the user can get the vision of the list of items based on their names after login in successfully. Detailed information about a particular item with the item name, item image, and item price can be seen by users. If the user like any item then the user needs to tap on that item to open item Details. On the Grocery Home screen, the user will also have an option “my orders” (basket logo at right top corner) to check orders.

This page has information regarding the grocery such as its name, image, desc small-screen-description, quantity and price information, etc. If the user wants to order the item then the user needs to click on the “Order Now” button And if the user wants to adjust the quantity of item then the user can adjust by clicking on-increment “+” and decrement “-” option.

'In My Orders Page, the user can view the list of items based on their orders after an order is ordered successfully. Brief information about a specific item with the item name, item image, and item price can be seen by users. Whenever a user orders any item from the Orders Detail page the order can be viewed in this module. Users can also delete specific orders by long-clicking the item. When the user long presses on the order, the pop-up screen occurs notifying “Are you sure you want to delete this item” and the positive “Yes” and Negative “No” options are available.
